Добавлены:
* авторизация,
* регистрация

Основной функционал разрабатывался в проекте https://github.com/ps-fedorova/mesto-react

## Локальная установка

1. git clone https://github.com/ps-fedorova/react-mesto-auth.git
2. cd react-mesto-auth
3. npm i

## Запуск проекта
npm start
